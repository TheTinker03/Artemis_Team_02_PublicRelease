WINDOWS INSTALL

I hate reading and I want a video tutorial: https://youtu.be/Bjkw375HnV0




Step One, install our app------------------------------------------------------------------------------------------

 a.Download our Game Coach setup exe from our git Link:github.com/TheTinker03/Artemis_Team_02_PublicRelease
 
 b.Find the exe file in the center of the screen and hit it, then hit the download button near the top right of your screen.

 c.After you downloaded it double click it.

 d.We didnt sign for our app so windows will pop up saying the app might be malicious, simply hit read more and run anyway.

 e.Run through the setup wizard.

 f.Select only for me

 g.Select location default is fine

 h.Hit finish

Step Two, install LMStudio-----------------------------------------------------------------------------------------

 a.Download the LMStudio installer Link: https://lmstudio.ai/

 b.Press the big download for windows button in the center of the screen.

 c.Run the installer exe

 d.Agree to the terms

 e.select only for me or Anyone depending if you want to run artemis on a different windows account.

 f.select a suitable location, default it usually fine.

 g.Hit Finish

Step Three, installing the ai--------------------------------------------------------------------------------------

 a.Run lm studio if its not already running

 b.If it asks you to install a model when opening it hit skip and continue

 c.Locate the model search button on the left side of the window.

 d.Search for Qwen3-vl 8b and select it from the top left

 e.Press the download button and wait for install

 f.After it has installed successfully find the developer tab on the left (should be above the model search tab)

 g.Select server settings at the top

 h.Enable the CORS option

 i.press out of the little pop up back to the developer tab

 j.press load model and select our qwen3 vl 8b model, wait for it to load

 k.Once it is loaded find the load tab to the TOP RIGHT (next to info and inference)

 l.Once in the load tab move the context length to 20000 (if your computer has issues go back by 5000 until it works)

 m.Reload the model by pressing the option at the bottom, if it has not popped up, eject the model and follow from step j again.

 n.once you have successfully loaded the model with a context length that is greater than 8000 return to the developer tab

 o.It is time to turn on the model, find the start server button (it should say status stopped as of now) and turn it on

Step Four, running the model---------------------------------------------------------------------------------------

 a.Open our app (Game Coach)

 b.Hit new user and enter your name

 c.Hit new session and name it what you want to and hit create.

 d.This is important, there is a box that says model at the bottom left, insert your model name there for you it is:qwen3-vl-8b

 e.You can now drag mp4's or press insert media and select a video

 f.you can now ask the ai about the video and it can respond warning this might take a while. The model might take a while you can see your response loading in lmstudio

 g.enjoy

INSTALL DONE!!!!



NORMAL RUNNING PROCESS

 a.Open lm studio and load the qwen model.

 b.adjust context length to known ammount (20,000).

 c.start server.

 d.open game coach app and enjoy.
